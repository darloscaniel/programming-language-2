# Programming Language II - Java Projects

This repository contains projects developed during the **Programming Language II** course, taught by **Professor Lucas Nadalete**. The course focuses on learning the basics of programming using **Java**.

## Course Overview

In this course, we explored the foundational concepts of Java programming, including:

- Object-oriented programming (OOP) principles (Classes, Objects, etc.)
- - Abstraction
- Classes and Objects
- Methods and Messages 
- Encapsulation
- Single and Multiple Inheritance 
- Polymorphism  
- Exception Handling  
- Data Input and Output
